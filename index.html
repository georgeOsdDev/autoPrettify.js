autoPrettify.js Sample

autoPrettify.js is a wrapper for prettify.{js|css}.

prettify.{js|css} is very good tool, bat it needs some step to highlight syntax. So autoPrettify.js help it's step automatically.


Usage

just include this script on head tag with this one line.

<script src="http://georgeosddev.github.com/autoPrettify.js/autoPrettify.min.js">

It automatically apply prettify.{js|css} at each <pre> tags and <code> tags with Adobe's Source Code Pro font.


Sample

<pre>Tag sample

import java.util.*
public static void main(String[] args){
  String msg = "Hello World!"
  System.out.println(msg);
}

<code>Tag sample

function foo(){ var msg = "Hello World!" console.log(msg); }
